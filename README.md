# Debouncer
Verilog deobuncer code that comes with 2 version

Debouncer can be use to eliminate unwanted bounces as an input square wave signal.

Unwanted bounces is mostly caused by mechanical switches.

Those bounces can be misinterpretated by microprocessors.

Version1 is more primal version. It comes with delay wich can cause problem in some operations which speed is important.

Version 2 does not have such a problem.
